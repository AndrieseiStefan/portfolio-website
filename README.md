# Personal Portfolio Website

A static personal portfolio website built with React, Vite, TypeScript, Tailwind CSS, and React Router.

## Stack

- React + Vite
- TypeScript
- Tailwind CSS
- React Router with `HashRouter` for GitHub Pages-friendly routing

## Available Pages

- Home
- About
- Services
- Portfolio
- Contact

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

The Vite `base` path is currently configured for a GitHub Pages repository deployment at `/portofolio-website/`.
